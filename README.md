Description

This project provides a simple SQL database template for managing employee data.
It is designed for learning purposes and can be used as a starter schema for HR or payroll systems.

The template includes basic fields such as employee name, salary, bonus, and department.

Database Structure
Employees Table
Column Name	Data Type	Description
id	INT	Unique employee ID
name	VARCHAR(225)	Employee full name
salary	INT	Employee salary
bonus	INT	Employee bonus

How to Use

Download the SQL file from this repository

Open MySQL, MariaDB, or phpMyAdmin

Import the SQL file

The database tables will be created automatically

Learning SQL and database design
